# Phoenix Travails – The Odin Project

This project is an interactive pathfinding visualization built as part of **The Odin Project** JavaScript curriculum.

It demonstrates how a knight (represented as a phoenix) finds the shortest path on a chessboard using the Breadth-First Search (BFS) algorithm.

---

## 📚 About The Project

This project focuses on solving the classic **Knight’s Travails** problem by modeling the chessboard as a graph and computing the shortest path between two positions.

The application allows users to:

- Select a starting position
- Select a target position
- Visualize the shortest path
- Watch an animated phoenix move across the board

The main goal was to strengthen understanding of:

- Graph traversal algorithms (BFS)
- Shortest path problem solving
- State management in JavaScript
- DOM manipulation and event handling
- UI interaction design

- Implementing Breadth-First Search (BFS)
- Modeling a chessboard as a graph
- Generating valid knight moves dynamically
- Tracking visited nodes efficiently using a Set
- Reconstructing shortest paths using parent references
- Structuring a modular JavaScript project
- Handling user interaction and UI state
- Creating responsive layouts using CSS Grid
- Integrating animations using `transform` and async logic
